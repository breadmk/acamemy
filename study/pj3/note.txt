스크래피 : 프레임워크

1)프로젝트 생성
scrapy startproject 프로젝트명
scrapy startproject find1   <-- 폴더가 생성됨.

---- 사용법

cd find1 -- scrapy.cfg 파일이 있는 곳에서 명령어 실행(terminal) 으로 이동
cd find1

2) 스파이더 생성 및 작성
---- 사용할 사이트를 http://제외하고 기록 (scrapy genspider example example.com)
(scrapy genspider 이름 수집할사이트)
scrapy genspider test1 10000recipe.com/

3) 실행
scrapy crawl 이름
scrapy crawl test1      --test1.py의 name값 (  name = 'test1' )
scrapy crawl test1 --nolog  --아무일 없이 실행.
