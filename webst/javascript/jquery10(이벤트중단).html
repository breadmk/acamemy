이벤트 처리막기
1) event.preventDefault() : 고유동작 중단
2) event.stopPropagation() : 이벤트가 상위 전파 중단
3) event.stopImmediatePropagation() : 상위뿐 아니라 같은 레벨로도 전파 중단
4) return false : event.stopPropagation() + event.preventDefault()
    jquery 사용시 : event.stopPropagation() + event.preventDefault() [고유동작+이벤트상위 중단]
    jquery 미사용시 : event.preventDefault() [고유동작 중단]
