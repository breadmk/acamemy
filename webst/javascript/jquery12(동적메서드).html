document객체 : html과 xml문서의 뼈대,문서의 구조나 내용에 접근할 수 있는 방법
Element : 하나의 태그 요소
Attribute : Element 에 속해 있는 속성
Node : Element 에서 시작되는 트리구조
next() : 다음
prev() : 이전
parent() : 부모
parents() : 조상
children() : 자손
find() : 찾기
eq(n) : 몇 번째 태그에 적용. 0부터 시작.

아이디:

비밀번호: 

bind() : $('선택자').bind(event,function(){})
         $('선택자').bind({'event1':function(){},
                         {'event2':function(){},
                         {'event3':function(){}
                         ... })
one() : 일회성 처리가 필요시, bind와 사용법 동일.
on() : 동적으로 추가할 요소에 대한 이벤트를 미리 정의해 놓는 기능.
       이벤트가 정의되는 시점에는 대상이 존재하지 않음.
       이 이벤트는 document 객체에 설정.
       $(document).on('event','선택자',function(){})
